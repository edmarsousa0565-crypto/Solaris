❌ Por Fazer — Crítico

Webhooks Stripe + CJ

Sem os webhooks, as encomendas não são criadas automaticamente no CJ após pagamento. É a peça mais crítica que falta.

Fluxo Automático

1 Cliente paga na SOLARIS (Stripe)
↓
2 Webhook Stripe → checkout.session.completed → cria encomenda no CJ
↓
3 Encomenda guardada na Supabase com status processing
↓
4 CJ processa e envia para o cliente
↓
5 Webhook CJ → status SHIPPED → Supabase atualizado → Resend envia email tracking
↓
6 Webhook CJ → status DELIVERED → email de entrega + pedido de review

Ficheiros a Criar

FicheiroFunçãoTrigger
app/api/webhooks/stripe/route.ts Recebe pagamento → cria encomenda CJ → guarda Supabase checkout.session.completed
app/api/webhooks/cj/route.ts Recebe update CJ → atualiza Supabase → email Resend Order status change

Tabela Supabase — orders

-- Executar no Supabase SQL Editor
create table orders (
  id              uuid default gen_random_uuid() primary key,
  stripe_session_id text unique not null,
  cj_order_id     text,
  customer_email  text not null,
  customer_name   text,
  shipping_address jsonb,
  items           jsonb,
  total_amount    numeric,
  currency        text default 'EUR',
  status          text default 'pending',
  tracking_number text,
  tracking_url    text,
  created_at      timestamptz default now(),
  updated_at      timestamptz default now()
);

Configurar URL no Stripe Dashboard

  • ○ stripe.com/dashboard → Developers → Webhooks → Add endpoint
  • ○ URL: https://solarisstore.pt/api/webhooks/stripe
  • ○ Eventos: checkout.session.completed + payment_intent.payment_failed
  • ○ Copiar Signing Secret → STRIPE_WEBHOOK_SECRET no .env.local
  • ○ CJ Dashboard → Settings → Webhook → URL: https://solarisstore.pt/api/webhooks/cj

Registo Legal em Portugal

Recomendado agora

Recibos Verdes

Trabalhador Independente
  • Custo: €0 para abrir
  • Online em 10 minutos
  • portaldasfinancas.gov.pt
  • CAE: 47910 (comércio internet)
  • Isento IVA até €13.500/ano
  • Sem contabilista obrigatório
Quando escalares

Empresa Lda

A partir de €2–3K/mês
  • Capital mínimo: €1
  • eportugal.gov.pt
  • Custo inicial ~€360
  • Contabilista ~€80–120/mês
  • Mais credibilidade

Atenção: Sem NIF ativo não podes emitir faturas — e em Portugal é obrigatório emitir fatura em todas as vendas online.

Checklist de Abertura

  • ○Aceder a portaldasfinancas.gov.pt com Chave Móvel Digital
  • ○Abrir atividade → CAE 47910
  • ○Selecionar regime de isenção art.º 53 (até €13.500/ano)
  • ○Atualizar páginas legais da SOLARIS com NIF
  • ○Configurar emissão de faturas (InvoiceXpress ou Moloni)
